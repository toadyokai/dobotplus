# 安全皮肤

> 不同版本的控制器、插件和上位机支持不同的 lua API 指令，开发者可在 DobotStudio Pro “应用” 菜单，“脚本编程” 的指令侧边栏查看具体支持的 lua API

## 指令列表

安全皮肤指令用于设置安全皮肤相关功能。

| 指令           | 功能                         |
| -------------- | ---------------------------- |
| EnableSafeSkin | 安全皮肤开关                 |
| SetSafeSkin    | 设置安全皮肤各个部位的灵敏度 |

## EnableSafeSkin

**原型：**

```lua
EnableSafeSkin(ON|OFF)
```

**描述：**

安全皮肤开关。

**必选参数：**

ON|OFF：ON 表示开启安全皮肤，OFF 表示关闭安全皮肤。

**返回：**

- 0：未检测到安全皮肤
- 1：检测到安全皮肤
  **示例：**

```lua
-- 开启安全皮肤。
EnableSafeSkin(ON)
```

## SetSafeSkin

**原型：**

```lua
SetSafeSkin(part,status)
```

**描述：**

设置安全皮肤各个部位的灵敏度。

**必选参数：**

- part：要设置的部位，取值范围 3~6
  - 3：小臂
  - 4：J4 关节
  - 5：J5 关节
  - 6：J:6 关节
- status：要设置的灵敏度，取值范围 0~3
  - 0：关闭
  - 1：低灵敏度（接近距离 ≤5cm）
  - 2：中灵敏度（接近距离 ≤10cm）
  - 3：高灵敏度（接近距离 ≤15cm）
    **示例：**

```lua
-- 将小臂的安全皮肤灵敏度设置为关闭。
SetSafeSkin(3,0)
```
